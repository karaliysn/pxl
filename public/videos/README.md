# Video Assets

Place your video files here:

- `hero-bg.mp4` — Hero section background video (looping, muted, autoplay)
  - Recommended: Dark, cinematic footage (landscape, city lights, motion blur, etc.)
  - Resolution: 1920×1080 or 3840×2160
  - Duration: 10–30 seconds (loops seamlessly)
  - Codec: H.264 / MP4 for broad browser support
  - File size: Keep under 20MB for fast loading

Reference in components with absolute path: `/videos/hero-bg.mp4`
